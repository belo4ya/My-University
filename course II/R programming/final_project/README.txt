Название сети магазинов - BSTBreadITWRLD

1. Исходные файлы данных: ./BSTBreadITWRLD/Магазин{n}/, n - номер магазина

2. Бат-файл для копирования данных: ./BSTBreadITWRLD/select.bat

3. Скрипты, считывающие данные, выполняющие расчеты, формирующие выходные 
файлы с результатами: ./analysisV2.R, ./analysis.R (устарело)

4. csv-файлы с результатами расчетов: ./BSTBreadITWRLD/Анализ/итог/*.csv, 
./BSTBreadITWRLD/Анализ/{store_name}/*.csv, store_name - название магазина, 
по которому проводился анализ

5. Скрипты и функции для генерации исходных данных: ./generate_data.R

6.1 Генерация графиков: ./visualizationV2.R
6.2 Сами графики: ./BSTBreadITWRLD/Анализ/итог/*.png, ./BSTBreadITWRLD/Анализ/{store_name}/*.png,
store_name - название магазина, по которому проводился анализ
